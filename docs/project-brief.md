# Idea Factory Project Brief

## Goal

Idea Factory collects product launch and market signal information, then turns it into structured startup idea candidates.

## Early Demo Scope

The first demo should stay simple:

1. Collect product information from Product Hunt or similar public sources.
2. Normalize each product into a structured record.
3. Generate idea candidates based on observed products, categories, pain points, and target users.
4. Save the output locally as JSON or Markdown.

## Non-goals for the First Demo

The first demo should not include:

- A web UI
- A database
- A complex multi-agent framework
- User accounts
- Payment
- Deployment automation

## Future Direction

Later versions may add:

- Multiple data sources
- Scheduled daily runs
- AI-based classification
- AI-based opportunity scoring
- Agent workflows
- Idea ranking
- Founder-fit filtering
